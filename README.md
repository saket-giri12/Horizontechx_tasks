# Sentiment Analysis
This project analyzes text and predicts whether the sentiment is Positive, Negative, or Neutral.

Features
- Text sentiment analysis
- Positive, Negative, and Neutral classification
- Simple data visualization

Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib
- Seaborn

 How to Run

Install the required libraries:

bash
pip install pandas textblob matplotlib seaborn


Run the project:

bash
python sentiment_analysis.py


# Language Translator

This project translates text from one language to another using Python.

Features
- Translate text into multiple languages
- Easy to use
- Fast translation

Technologies Used
- Python
- Googletrans

 How to Run

Install the required library:

bash
pip install googletrans==4.0.0-rc1


Run the project:

bash
python translator.py
